# preload.js

## Описание

Предзагрузочный скрипт Electron приложения Likbez. Выполняется в контексте веб-страницы до её загрузки и отвечает за инициализацию API моста для безопасного взаимодействия между renderer и main процессами.

## Основные функции

### Инициализация API моста

Создает экземпляр класса APIBridge, который предоставляет безопасный интерфейс для взаимодействия с основным процессом приложения.

## Безопасность

Предзагрузочный скрипт обеспечивает безопасность приложения путем:

- Использования контекстной изоляции (contextIsolation)
- Предоставления ограниченного API через Context Bridge
- Отсутствия прямого доступа к Node.js API из веб-страницы

## Импорты

- APIBridge: класс для создания безопасного API моста между процессами
