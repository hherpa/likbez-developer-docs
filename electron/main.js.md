# main.js

## Описание

Точка входа в Electron приложение Likbez. Этот файл отвечает за инициализацию приложения, создание главного окна, регистрацию кастомного протокола и настройку обработчиков IPC сообщений.

## Основные функции

### createMainWindow()

Создает главное окно приложения с заданными параметрами:

- Размеры: 1280x800 (минимальный размер 1024x640)
- Автоматическое скрытие меню на Windows
- Включенная контекстная изоляция для безопасности
- Отключенная интеграция Node.js
- Установленный путь к preload.js

В режиме разработки загружает контент с локального сервера и открывает инструменты разработчика. В production режиме загружает статические файлы из папки build.

### Обработчики событий приложения

- `app.on('ready')` - инициализирует обработчики и создает главное окно
- `app.on('activate')` - создает новое окно при активации (macOS)
- `app.on('window-all-closed')` - завершает работу приложения при закрытии всех окон

## Регистрация кастомного протокола

Регистрирует кастомный протокол 'likbez' с привилегиями:
- standard: true (стандартный протокол)
- secure: true (безопасный протокол)
- supportFetchAPI: true (поддержка Fetch API)
- stream: true (поддержка потоков)
- corsEnabled: true (включенный CORS)

## Импорты

- electron: основные модули Electron (app, BrowserWindow, protocol)
- path: модуль для работы с путями файловой системы
- ProtocolHandler: обработчик кастомного протокола
- IPCHandlers: обработчики IPC сообщений
