# FlowContainer.js

## Обзор

Компонент-контейнер для диаграммы ReactFlow. Предоставляет базовую структуру и стилизацию для отображения диаграммы узлов.

## Назначение

FlowContainer - это простой контейнерный компонент, который:
- Занимает 100% высоты и ширины родительского элемента
- Передает все props дочерним элементам
- Предоставляет гибкую стилизацию через style prop

## Свойства (Props)

### children (ReactNode)
Дочерние элементы, которые будут отображаться внутри контейнера. Обычно это компонент ReactFlow с диаграммой.

### style (Object, optional)
Дополнительные стили для контейнера. Объединяются с базовыми стилями (100% высоты и ширины).

## Стилизация

Компонент использует inline-стили для установки базовых размеров:
```css
{
  height: '100%',
  width: '100%',
  ...style // дополнительные стили от props
}
```

## Использование

Компонент используется в основном компоненте App.js как контейнер для ReactFlow:

```jsx
<FlowContainer>
  <ReactFlow
    nodes={nodes}
    edges={edges}
    // ... другие props
  >
    {/* Дочерние компоненты ReactFlow */}
  </ReactFlow>
</FlowContainer>
```

## Зависимости

- React (для создания компонента)
- Нет внешних зависимостей

## Принципы работы

1. **Простота** - минимальная функциональность, только контейнер
2. **Гибкость** - позволяет передавать дополнительные стили
3. **Совместимость** - работает с любыми дочерними элементами
