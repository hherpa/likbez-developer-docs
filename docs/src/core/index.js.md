# index.js

## Обзор

Этот файл является точкой входа для модуля `core`, экспортируя все основные компоненты, провайдеры и хуки, которые используются в приложении. Он объединяет функциональность из различных подмодулей `app`, `providers` и `hooks`.

## Экспорты

### Компоненты приложения
```javascript
export * from './app';
```
Экспортирует все компоненты из подмодуля `app`, включая главный компонент `App`, контейнер для диаграммы `FlowContainer` и константы приложения.

### Провайдеры
```javascript
export * from './providers';
```
Экспортирует все React Context провайдеры, которые управляют глобальным состоянием приложения:
- `AppProvider` - управление тегами категорий и глобальным состоянием
- `ElectronProvider` - интеграция с Electron API
- `FlowProvider` - состояние узлов и связей в диаграмме
- `NavigationProvider` - состояние навигации между уровнями

### Хуки по категориям
```javascript
export * from './hooks/diagram';
export * from './hooks/navigation';
export * from './hooks/storage';
```
Экспортирует кастомные хуки, организованные по функциональным категориям:
- `hooks/diagram` - хуки для работы с диаграммой узлов (drag & drop, навигация)
- `hooks/navigation` - хуки для навигации и управления UI
- `hooks/storage` - хуки для работы с данными и состоянием

## Использование

Основной способ использования экспортированных модулей:

```javascript
// Импорт всех компонентов app
import { App, FlowContainer } from '@/core';

// Импорт провайдеров
import { AppProvider, FlowProvider } from '@/core';

// Импорт хуков по категориям
import { useFlow, useNodeEditor } from '@/core';
import { useNavigation, useUI } from '@/core';
import { useTheme, useDataPersistence } from '@/core';
```

## Зависимости

Файл зависит от:
- Подмодулей `./app`, `./providers` и `./hooks/*`
- React и React Hooks

## Принципы организации

1. **Единая точка входа** - все основные экспорты доступны через один файл
2. **Логическая группировка** - компоненты сгруппированы по функциональности
3. **Масштабируемость** - легко добавлять новые экспорты в соответствующие категории
