# README.md

## Назначение
Документация для папки редактируемых компонентов в визуальном редакторе, описывающая структуру, файлы и архитектуру компонентов для редактирования контента в превью режиме.

## Экспортируемые компоненты
- Нет экспортируемых компонентов (документация)

## Структура
1. Заголовок и описание папки
2. Описание файлов:
   - `EditableContentRenderer.js` - рендерер для редактируемого контента
   - `EditableElement.js` - главный компонент для редактируемого элемента
   - `EditableContent.js` - компонент для рендеринга contentEditable
   - `ContentEditableUtils.js` - утилиты для работы с contentEditable элементами
3. Архитектура компонентов
4. Зависимости между компонентами

## Взаимодействия
1. Описывает взаимодействие между компонентами редактирования
2. Объясняет ответственность каждого компонента
3. Показывает архитектурные связи между файлами
4. Описывает зависимости компонентов

## Зависимости
- Нет внешних зависимостей (документация)

## Архитектура
Описывает архитектуру редактируемых компонентов:

```
EditableContentRenderer
    ↓
EditableElement (координатор)
    ↓
EditableContent (рендеринг contentEditable)
    ↓
ContentEditableUtils (утилиты)
```

Документация описывает:
1. `EditableContentRenderer` - рендерер для редактируемого контента в превью, обертывающий элементы в редактируемые компоненты
2. `EditableElement` - главный компонент для редактируемого элемента, координирующий работу подкомпонентов
3. `EditableContent` - компонент для рендеринга contentEditable элемента
4. `ContentEditableUtils` - утилиты для работы с contentEditable элементами

Также описывает зависимости:
- `EditableContentRenderer` использует `EditableElement`
- `EditableElement` использует `EditableContent` и `ContentEditableUtils`
- Все компоненты используют систему рендеринга и утилиты из `../system/` и `../utils/`
