# TextRemover.js

## Назначение
Модуль для удаления выделенного текста из markdown в редакторе, отвечающий за удаление текста из строк markdown.

## Экспортируемые компоненты
- `removeSelectedTextFromMarkdown` - функция удаления выделенного текста из строки markdown

## Структура
1. Импорты TextFinder и логгера
2. Функция `removeSelectedTextFromMarkdown` для удаления текста из markdown

## Взаимодействия
1. Использует TextFinder для поиска текста в строке
2. Использует логгер для отладки процесса удаления
3. Работает со строками markdown
4. Принимает строку markdown и выделенный текст
5. Возвращает результат удаления с новой строкой

## Зависимости
- `TextFinder.js` - для поиска текста в строке
- `Logger.js` - для логирования

## Архитектура
Следует принципу единственной ответственности (SRP), отвечая только за удаление текста.

Функция `removeSelectedTextFromMarkdown` реализует логику:
1. Проверяет валидность входных параметров
2. Удаляет пробелы по краям у выделенного текста
3. Извлекает префикс из строки markdown (для заголовков)
4. Удаляет префикс из строки для поиска
5. Логирует процесс удаления
6. Проверяет, совпадает ли выделенный текст со всей строкой
7. Ищет текст в строке через findTextInLine
8. Удаляет текст из строки через removeTextFromLine
9. Обрабатывает случаи пустой строки после удаления
10. Возвращает результат удаления с новой строкой и префиксом
