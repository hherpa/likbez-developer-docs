# ListUpdater.js

## Назначение
Вспомогательные функции для обновления списков в редакторе, обеспечивающие обработку и обновление строк списков.

## Экспортируемые компоненты
- `updateListLines` - функция обновления строк списков

## Структура
1. Функция `updateListLines` для обновления строк списков

## Взаимодействия
1. Принимает новый текст, массив строк и индекс целевой строки
2. Разбивает новый текст на строки
3. Добавляет маркеры списков к строкам
4. Находит конец списка в оригинальных строках
5. Заменяет строки списка в массиве

## Зависимости
- Нет внешних зависимостей

## Архитектура
Следует принципу единственной ответственности (SRP), отвечая только за обновление списков.

Функция `updateListLines` реализует логику:
1. Разбивает новый текст на строки и фильтрует пустые
2. Добавляет маркеры списков к строкам:
   - Если строка уже начинается с '-', оставляет как есть
   - Иначе добавляет '- ' в начало
3. Находит конец списка в оригинальных строках:
   - Начинает с целевой строки
   - Продолжает, пока следующая строка начинается с '- '
4. Заменяет строки списка в массиве:
   - Удаляет старые строки списка
   - Вставляет новые строки с маркерами
