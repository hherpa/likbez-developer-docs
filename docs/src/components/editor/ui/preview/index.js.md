# index.js

## Описание
Файл экспорта компонентов превью. Служит единой точкой входа для всех модулей preview.

## Экспортируемые компоненты
- `EditorFlowPreview` - компонент превью редактора

## Структура
- Экспорт превью компонентов:
  - `EditorFlowPreview` из `./EditorFlowPreview`

## Взаимодействия
- Использует компонент `EditorFlowPreview` из файла `./EditorFlowPreview`

## Зависимости
- `./EditorFlowPreview` - компонент превью редактора

## Архитектура
Файл служит точкой входа для модуля preview, обеспечивая централизованный экспорт всех необходимых компонентов превью. Это позволяет использовать компоненты превью через единый импорт.
