# EditorIcons Component Documentation

## Описание
EditorIcons - это компонент, который предоставляет коллекцию SVG иконок для использования в редакторе. Компонент экспортирует объект с различными иконками, которые могут быть использованы в пользовательском интерфейсе редактора. Все иконки реализованы как функциональные компоненты React, которые возвращают SVG элементы с настраиваемыми размерами и классами.

## Экспортируемые компоненты
- `EditorIcons` - объект, содержащий все иконки редактора
- `ArrowRight` - иконка стрелки вправо
- `Bold` - иконка жирного текста
- `Italic` - иконка курсивного текста
- `Code` - иконка кода
- `List` - иконка списка
- `HR` - иконка горизонтальной линии
- `Link` - иконка ссылки
- `Image` - иконка изображения
- `Tag` - иконка тега
- `ChevronDown` - иконка стрелки вниз
- `Handle` - иконка ручки
- `H1` - иконка заголовка H1
- `H2` - иконка заголовка H2
- `H3` - иконка заголовка H3
- `Name` - иконка имени
- `Latex` - иконка LaTeX
- `BoxLatex` - иконка блочного LaTeX
- `Preview` - иконка предварительного просмотра
- `SplitView` - иконка разделенного просмотра
- `Eye` - иконка глаза (видимость)
- `EyeOff` - иконка перечеркнутого глаза (невидимость)
- `Flow` - иконка потока
- `Content` - иконка контента

## Структура компонента
Компонент представляет собой объект `EditorIcons`, который содержит множество функциональных компонентов, каждый из которых возвращает SVG элемент. Каждая иконка принимает пропсы `size` (размер иконки, по умолчанию 16) и `className` (дополнительные CSS классы). Все иконки реализованы с использованием SVG элементов с соответствующими атрибутами для отображения нужной графики.

## Взаимодействия
Каждая иконка реализована как функциональный компонент React, который принимает пропсы `size` и `className`. Иконки не имеют внутреннего состояния и не реализуют сложных взаимодействий. Они просто возвращают SVG элементы с заданными параметрами. Все иконки используют `currentColor` для окраски, что позволяет наследовать цвет от родительского элемента.

## Зависимости
- React (для создания функциональных компонентов)
- SVG (для отображения графики)

## Архитектура
Компонент следует паттерну коллекции иконок, где все иконки собраны в одном объекте для удобного импорта и использования. Каждая иконка реализована как отдельная функция, что обеспечивает модульность и переиспользуемость. Использование пропсов `size` и `className` обеспечивает гибкость настройки внешнего вида иконок. Все иконки используют `currentColor` для окраски, что позволяет легко адаптировать их под различные темы и стили. Такая архитектура обеспечивает легкость поддержки и расширения коллекции иконок. Компонент экспортируется по умолчанию, что позволяет легко импортировать его в других частях приложения. Использование SVG обеспечивает масштабируемость иконок без потери качества. Все иконки имеют четко определенную структуру с viewBox="0 0 24 24", что обеспечивает единообразие отображения.
