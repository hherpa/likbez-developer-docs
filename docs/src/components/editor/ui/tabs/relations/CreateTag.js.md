# CreateTag Component Documentation

Компонент `CreateTag` предоставляет интерфейс для создания новых тегов.

## Основные функции

- Отображает поле ввода для нового тега
- Предоставляет кнопку для создания тега

## Структура компонента
1. **Поле ввода**:
   - input элемент для ввода названия тега
|---------|-----|-------------|----------|
   - стилизация с закругленными углами и границей

2. **Кнопка создания**:
   - button элемент для подтверждения создания тега
   - текст "Создать"
   - стилизация с закругленными углами и границей

3. **Контейнер**:
   - flex контейнер для расположения элементов ввода и кнопки
   - отступ между элементами

## Взаимодействия
Компонент использует локальное состояние через `useState` для управления значением поля ввода. При нажатии на кнопку "Создать" компонент:
1. Проверяет, что введено непустое значение (после trim)
2. Вызывает переданный коллбэк `onCreate` с введенным значением
3. Очищает поле ввода

Компонент также поддерживает ввод с клавиатуры через стандартное поведение input элемента. При нажатии Enter в поле ввода будет срабатывать стандартное событие формы (если она есть), либо нужно добавить обработчик onKeyDown.

## Зависимости
- React (useState)
- CSS (для стилизации элементов)

## Архитектура
Компонент следует паттерну контролируемого компонента React, где состояние поля ввода управляется через React состояние. Компонент инкапсулирует логику валидации введенного значения и очистки поля после создания тега. Используется функциональный подход с хуками для управления состоянием. Компонент следует принципу единственной ответственности - он только отвечает за создание тегов и передачу их родительскому компоненту. Стилизация компонента реализована inline стилями, что обеспечивает независимость от внешних CSS файлов. Компонент экспортируется по умолчанию, что упрощает его импорт в других частях приложения. Все обработчики событий мемоизированы неявно через замыкания, что предотвращает создание новых функций при каждом рендере. Компонент не имеет сложных побочных эффектов и зависит только от переданных пропсов.
