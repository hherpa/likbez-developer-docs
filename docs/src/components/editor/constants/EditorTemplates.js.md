# Документация EditorTemplates.js

## 1. Назначение файла

Файл src/components/editor/constants/EditorTemplates.js содержит константы шаблонов для вставки различных элементов в редактор. Эти шаблоны используются для стандартизации форматирования и упрощения вставки элементов синтаксиса в текст редактора.

## 2. Экспортируемые константы и их структура

Файл экспортирует три основные константы:

### DEFAULT_TEXTS
Объект, содержащий тексты по умолчанию для различных элементов:
- name: 'Имя узла'
- tag: 'Новый тег'
- link: 'node_id'
- symbol: 'A, B, C'
- handle: 'Новый handle'
- image: 'URL_изображения'
- latex: 'x^2 + y^2 = z^2'
- 'box-latex': 'E = mc^2'
- colorText: 'цветной текст'
- bgColorText: 'текст с фоном'
- h1: 'Заголовок 1'
- h2: 'Заголовок 2'
- h3: 'Заголовок 3'
- bold: 'жирный текст'
- italic: 'курсивный текст'
- code: 'код'
- linkText: 'текст ссылки'
- list: 'элемент списка'
- quote: 'цитата'

### TAG_TEMPLATES
Объект, содержащий функции для создания шаблонов элементов синтаксиса:
- name: {.name} content {.name}
- tag: {.tag} content {.tag}
- link: {.link} content {.link}
- symbol: {.symbol} content {.symbol}
- handle: {.handle} content {.handle}
- image: {.image} content {.image}
- latex: {.LaTeX} content {.LaTeX}
- 'box-latex': {.box-LaTeX} content {.box-LaTeX}
- color: {.color} content {.color}
- bgColor: {.bg-color} content {.bg-color}

### MARKDOWN_TEMPLATES
Объект, содержащий функции для создания шаблонов Markdown:
- h1: # text
- h2: ## text
- h3: ### text
- bold: **text**
- italic: *text*
- code: `text`
- link: [text](URL)
- list: - text
- quote: > text
- hr: ---

## 3. Использование конфигурации в редакторе

Константы из этого файла используются в редакторе для:
1. Предоставления текстов по умолчанию при вставке элементов
2. Форматирования элементов синтаксиса согласно заданному шаблону
3. Создания Markdown разметки при использовании соответствующих инструментов

## 4. Взаимодействие с другими компонентами

Этот файл используется различными компонентами редактора, включая:
- Компоненты панели инструментов для вставки элементов
- Обработчики элементов для форматирования контента
- Компоненты вставки Markdown

## 5. Используемые зависимости

Файл не имеет внешних зависимостей и представляет собой чистый JavaScript модуль с константами.
