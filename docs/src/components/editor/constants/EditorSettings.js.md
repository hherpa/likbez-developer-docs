# Документация для EditorSettings.js

## 1. Назначение файла

Файл src/components/editor/constants/EditorSettings.js содержит конфигурационные константы, определяющие настройки редактора узлов. Основное назначение - предоставление параметров для управления размерами узлов в редакторе.

## 2. Экспортируемые константы

### NODE_SIZE_CONFIG
Объект, содержащий настройки размера узла:

```javascript
{
  minWidth: 200,        // Минимальная ширина узла в пикселях
  maxWidth: 800,          // Максимальная ширина узла в пикселях
  defaultWidth: 300,      // Ширина узла по умолчанию в пикселях
  label: 'Минимальная ширина',     // Метка для отображения в интерфейсе
  placeholder: '200',              // Заполнитель для поля ввода
  hint: 'Узел может расширяться при необходимости'  // Подсказка для пользователя
}
```

## 3. Использование конфигурации в редакторе

Конфигурация используется для:
- Определения границ размеров узлов (минимальная, максимальная, по умолчанию)
- Отображения соответствующих элементов управления в интерфейсе настроек редактора
- Обеспечения валидации вводимых пользователем значений размеров

## 4. Взаимодействие с другими компонентами

Константы из этого файла могут использоваться в:
- Компонентах настроек редактора (EditorSettingsTab)
- Хуках управления размерами узлов
- Компонентах пользовательского интерфейса, отвечающих за отображение и редактирование размеров

## 5. Используемые зависимости

Файл не содержит внешних зависимостей и представляет собой чистый JavaScript модуль с именованным экспортом константы.
