# EditorButtons.js

## Назначение файла

Файл [`src/components/editor/constants/EditorButtons.js`](file:///C:/Users/Akidra/Desktop/likbez_code/src/components/editor/constants/EditorButtons.js) содержит конфигурационные данные для кнопок и вкладок редактора. Он определяет структуру элементов интерфейса, которые будут отображаться в панели инструментов редактора, включая кнопки форматирования текста (Markdown), специализированные элементы (символы, изображения, LaTeX) и вкладки редактора.

## Экспортируемые константы и их структура

Файл экспортирует три основные константы:

### MARKDOWN_BUTTONS_CONFIG
Массив объектов, определяющих кнопки форматирования текста в стиле Markdown:
- `type`: Тип форматирования (например, 'h1', 'bold', 'italic')
- `label`: Отображаемый текст на кнопке

### CUSTOM_NODE_BUTTONS_CONFIG
Массив объектов, определяющих кнопки для вставки специализированных элементов:
- `type`: Тип элемента (например, 'symbol', 'image', 'latex')
- `label`: Отображаемый текст на кнопке

### EDITOR_TABS_CONFIG
Массив объектов, определяющих вкладки редактора:
- `id`: Идентификатор вкладки
- `label`: Отображаемый текст вкладки

## Использование кнопок в редакторе

Кнопки из этих конфигураций используются для создания панели инструментов редактора, позволяя пользователям быстро вставлять соответствующие элементы форматирования или специализированные компоненты в редактируемый контент.

## Взаимодействие с другими компонентами

Конфигурации из этого файла используются компонентами редактора для:
- Отображения панели инструментов
- Определения доступных операций форматирования
- Создания пользовательского интерфейса вкладок редактора

## Используемые зависимости

Файл не содержит внешних зависимостей и представляет собой чистый JavaScript модуль с константами.
