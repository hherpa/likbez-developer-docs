# Документация для файла EditorUI.js

## 1. Назначение файла

Файл src/components/editor/constants/EditorUI.js содержит UI константы, используемые в редакторе узлов приложения. Эти константы определяют текстовые значения и сообщения, отображаемые в пользовательском интерфейсе редактора.

## 2. Экспортируемые константы и их структура

Файл экспортирует следующие константы:

### CONFIRM_DIALOG
Объект, содержащий тексты для диалога подтверждения закрытия редактора с несохраненными изменениями:
- unsavedChanges.title: Заголовок диалога ("Несохраненные изменения")
- unsavedChanges.message: Сообщение диалога ("У вас есть несохраненные изменения. Закрыть без сохранения?")
- unsavedChanges.confirmText: Текст кнопки подтверждения ("Закрыть")
- unsavedChanges.cancelText: Текст кнопки отмены ("Отмена")

### TEXTAREA_PLACEHOLDER
Строка с placeholder текстом для textarea редактора:
- "Введите содержимое узла используя markdown синтаксис..."

### ACTION_BAR
Объект с текстами для action bar редактора:
- shortcutsInfo: Информация о горячих клавишах ("Ctrl+S для сохранения, Esc для отмены")
- saveButton: Текст кнопки сохранения ("Сохранить")
- cancelButton: Текст кнопки отмены ("Отмена")

### PREVIEW
Объект с текстами для режима предварительного просмотра:
- title: Заголовок предварительного просмотра ("Предварительный просмотр")

## 3. Использование конфигурации в редакторе

Константы из этого файла используются в различных компонентах редактора для отображения текстов и сообщений пользователю. Например:
- CONFIRM_DIALOG используется в EditorConfirmDialog для отображения диалога подтверждения
- TEXTAREA_PLACEHOLDER используется в EditorTextarea как placeholder
- ACTION_BAR используется в EditorActionBar для отображения текстов кнопок и информации о горячих клавишах

## 4. Взаимодействие с другими компонентами

Константы из этого файла импортируются и используются в следующих компонентах:
- EditorConfirmDialog - для отображения диалога подтверждения
- EditorTextarea - для отображения placeholder текста
- EditorActionBar - для отображения текстов кнопок и информации о горячих клавишах
- EditorFlowPreview - для отображения заголовка предварительного просмотра

## 5. Используемые зависимости

Файл не имеет внешних зависимостей и экспортирует константы напрямую. Он представляет собой простой JavaScript модуль с экспортами констант, который может быть импортирован в других частях приложения.
