# Документация для EditorDropdowns.js

## 1. Назначение файла

Файл src/components/editor/constants/EditorDropdowns.js содержит константы, используемые для настройки dropdown (выпадающих списков) в редакторе. Эти константы определяют описания элементов и заголовки для различных dropdown компонентов интерфейса редактора.

## 2. Экспортируемые константы и их структура

### ELEMENT_DESCRIPTIONS
Объект, содержащий описания для элементов dropdown меню:
- symbol: 'Добавить символ'
- handle: 'Добавить handle'
- image: 'Добавить изображение'
- latex: 'Добавить LaTeX формулу'
- box-latex: 'Добавить LaTeX в рамке'

### DROPDOWN_HEADERS
Объект, содержащий заголовки для dropdown меню:
- color: 'Цвет текста'
- backgroundColor: 'Цвет фона'

## 3. Использование конфигурации в редакторе

Константы из этого файла используются в компонентах редактора для отображения понятных пользователю названий элементов и заголовков в dropdown меню. Они обеспечивают локализацию интерфейса редактора на русский язык.

## 4. Взаимодействие с другими компонентами

Эти константы могут использоваться в следующих компонентах и хуках:
- Компоненты UI редактора, отвечающие за отображение dropdown меню
- Хуки управления dropdown меню (например, useDropdown)
- Компоненты панели инструментов редактора

## 5. Используемые зависимости

Файл не содержит внешних зависимостей и представляет собой чистый JavaScript модуль с экспортами констант.
