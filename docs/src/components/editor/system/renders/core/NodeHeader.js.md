# NodeHeader.js

## Описание
NodeHeader - компоненты для рендеринга заголовка узла и пустого узла.

## Экспортируемые компоненты
- `renderHeader(data, parsed, context, config, styles)` - функция для рендеринга заголовка узла
- `renderEmptyNode(context, config, styles)` - функция для рендеринга пустого узла

## Структура
- Функция `renderHeader` принимает данные узла, результат парсинга, контекст, конфигурацию и стили
- Функция `renderEmptyNode` принимает контекст, конфигурацию и стили

## Взаимодействие
- Использует `parseMarkdownContent` из '../../parser' для парсинга контента
- Получает цвет текста из конфигурации
- Получает стили заголовка из стилей
- Получает темные цвета из стилей

## Зависимости
- `react` для создания компонентов
- `../../parser` для парсинга контента

## Архитектура
Функции `renderHeader` и `renderEmptyNode` реализуют рендеринг заголовка узла и пустого узла соответственно. Функция `renderHeader` извлекает заголовок из результата парсинга и применяет соответствующие стили. Функция `renderEmptyNode` отображает сообщение о пустом узле с соответствующими стилями.
