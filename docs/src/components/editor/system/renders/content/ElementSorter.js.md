# ElementSorter.js

## Описание
ElementSorter - сортировка элементов. Отвечает только за сортировку элементов по позиции.

## Экспортируемые компоненты
- `ElementSorter` - сортировщик элементов

## Структура
- `sortElementsByPosition(elements)` - сортирует элементы по позиции
- `sortElementsByType(elements)` - сортирует элементы по типу
- `sortElements(elements)` - сортирует элементы по позиции и типу

## Взаимодействия
- Принимает массив элементов и сортирует их по различным критериям
- Не имеет внешних зависимостей

## Зависимости
- Нет внешних зависимостей

## Архитектура
Сортировщик элементов реализует паттерн "Стратегия" для различных алгоритмов сортировки:
1. Метод `sortElementsByPosition` реализует сложную логику сортировки:
   - Разделяет элементы на markdown и блочные
   - Сортирует параграфы по позиции
   - Для элементов разных типов определяет ближайший параграф
   - Обеспечивает правильный порядок отображения элементов
2. Метод `sortElementsByType` сортирует элементы по заранее определенному порядку типов
3. Метод `sortElements` является фасадом, использующим основной алгоритм сортировки

Класс следует принципу единственной ответственности, отвечая только за сортировку элементов. Поддерживает различные стратегии сортировки:
- По позиции (основная)
- По типу
- Комбинированная

Архитектура позволяет легко расширять поддержку новых стратегий сортировки, добавляя новые методы. Логика сортировки по позиции учитывает сложные случаи, когда блочные элементы должны отображаться рядом с определенными параграфами.
