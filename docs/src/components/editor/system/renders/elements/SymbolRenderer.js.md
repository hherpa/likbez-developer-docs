# SymbolRenderer.js

## Описание
SymbolRenderer - рендерер для символов. Соблюдает SOLID принципы.

## Экспортируемые компоненты
- `SymbolRenderer` - рендерер символов
- `SymbolRendererComponent` - функциональный компонент для обратной совместимости

## Структура
- `constructor()` - инициализация рендерера для типа 'symbol'
- `renderElement(element, context)` - рендерит символы

## Взаимодействия
- Наследуется от BaseRenderer
- Использует React для рендеринга элементов

## Зависимости
- `react` - для рендеринга React элементов
- `../core/BaseRenderer` - базовый класс рендерера

## Архитектура
Рендерер символов реализует паттерн "Коллекция" для отображения группы символов:
1. Метод `renderElement` принимает элемент и контекст рендеринга
2. Поддерживает разные форматы данных (symbols, content)
3. Рендерит символы в виде горизонтального списка с отступами
4. Каждый символ отображается в виде стилизованного блока:
   - Жирный шрифт размером 16px
   - Цвет текста и фона в зависимости от темы
   - Рамка и скругленные углы
   - Минимальная ширина для равномерного отображения
5. Поддерживает темизацию через контекст

Функциональный компонент `SymbolRendererComponent` обеспечивает обратную совместимость для использования в JSX.

Класс следует принципу единственной ответственности, отвечая только за рендеринг символов. Поддерживает гибкий формат данных и адаптивное отображение.
