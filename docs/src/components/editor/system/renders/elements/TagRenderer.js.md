# TagRenderer.js

## Описание
TagRenderer - рендерер для тегов. Соблюдает SOLID принципы.

## Экспортируемые компоненты
- `TagRenderer` - класс рендерера для тегов
- `TagRendererComponent` - функциональный компонент для обратной совместимости

## Структура
- Класс `TagRenderer`, наследующий `BaseRenderer`
- Метод `renderElement` для рендеринга тегов
- Функциональный компонент `TagRendererComponent` для обратной совместимости

## Взаимодействие
- Наследует от `BaseRenderer` и реализует метод `renderElement`
- Использует `getThemeColors` для получения цветов темы
- Принимает элемент и контекст для рендеринга

## Зависимости
- `react` для создания компонентов
- `BaseRenderer` из '../core/BaseRenderer'

## Архитектура
Класс `TagRenderer` наследует от `BaseRenderer` и реализует метод `renderElement` для рендеринга тегов. Компонент создает span элемент с определенными стилями для отображения тега, включая фон, цвет текста, отступы, скругленные углы, размер шрифта и границу. Также экспортируется функциональный компонент `TagRendererComponent` для обратной совместимости, который создает экземпляр `TagRenderer` и вызывает его метод `renderElement`.
