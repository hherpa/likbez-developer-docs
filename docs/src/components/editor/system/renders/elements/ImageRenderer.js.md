# ImageRenderer.js

## Описание
ImageRenderer - рендерер для изображений. Соблюдает SOLID принципы.

## Экспортируемые компоненты
- `ImageRenderer` - класс рендерера для изображений
- `ImageRendererComponent` - функциональный компонент для обратной совместимости

## Структура
- Класс `ImageRenderer`, наследующий `BaseRenderer`
- Метод `renderElement` для рендеринга изображений
- Функция `handleImageError` для обработки ошибок загрузки изображений
- Функциональный компонент `ImageRendererComponent` для обратной совместимости

## Взаимодействие
- Наследует от `BaseRenderer` и реализует метод `renderElement`
- Обрабатывает локальные пути data:// и Data:// (для обратной совместимости)
- Принимает элемент и контекст для рендеринга

## Зависимости
- `react` для создания компонентов
- `BaseRenderer` из '../core/BaseRenderer'

## Архитектура
Класс `ImageRenderer` наследует от `BaseRenderer` и реализует метод `renderElement` для рендеринга изображений. Компонент обрабатывает локальные пути к изображениям, заменяя префиксы data:// и Data:// на ./data/. Реализована обработка ошибок загрузки изображений с отображением placeholder'а. Также экспортируется функциональный компонент `ImageRendererComponent` для обратной совместимости, который создает экземпляр `ImageRenderer` и вызывает его метод `renderElement`.
