# ContentCleaner.js - Документация

## Общее описание
Файл src/components/editor/system/parser/content/ContentCleaner.js содержит класс ContentCleaner, который отвечает исключительно за очистку текстового контента от специальных тегов и форматирования, используемых в редакторе.

## Класс ContentCleaner

### Методы

#### cleanContent(content)
Очищает контент от специальных тегов элементов, оставляя только содержимое.

**Параметры:**
- content (string): Исходный контент для очистки

**Возвращает:**
- string: Очищенный контент

**Процесс очистки:**
1. Удаляет теги цветов, оставляя только содержимое (паттерны colorText и bgColor из ParserConfig.js)
2. Удаляет остальные теги элементов, оставляя содержимое (паттерны image, handle, symbol, element, link, name)
3. Очищает от лишних пробелов и множественных переносов строк

#### cleanLatexDuplicates(content)
Очищает контент от дубликатов LaTeX тегов.

**Параметры:**
- content (string): Исходный контент для очистки

**Возвращает:**
- string: Контент без дубликатов LaTeX тегов

**Процесс очистки:**
1. Находит все LaTeX теги ({.LaTeX}...{.LaTeX}) и теги Box-LaTeX ({.box-LaTeX}...{.box-LaTeX})
2. Удаляет дубликаты, оставляя только первые вхождения
3. Использует Set для отслеживания уже встречавшихся значений

## Зависимости
- src/components/editor/system/parser/core/ParserConfig.js - Использует паттерны и настройки для очистки контента

## Использование
Класс используется в системе парсинга редактора для подготовки "чистого" текста, например, для отображения в превью или для экспорта контента без форматирования.
