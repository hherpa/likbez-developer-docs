# MarkdownParserUtils.js

Утилиты для парсинга Markdown контента.

## Функции

### parseMarkdownBlocks(text, parseFn)

Разбирает текст на блоки, разделенные горизонтальными линиями (---).

**Параметры:**
- `text` (string): Текст для парсинга
- `parseFn` (function): Функция для парсинга отдельных блоков

**Возвращает:**
- `Array`: Массив элементов блоков

### processInlineMarkdown(text)

Обрабатывает встроенный Markdown (жирный, курсив, код).

**Параметры:**
- `text` (string): Текст для обработки

**Возвращает:**
- `Object`: Объект с типом и содержимым элемента
