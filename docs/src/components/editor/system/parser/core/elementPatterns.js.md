# elementPatterns.js

## Общее описание
Файл `elementPatterns.js` содержит набор простых функций для определения типов элементов в тексте. Эти функции используются при парсинге для идентификации различных специальных элементов, таких как LaTeX, изображения, теги и другие.

## Функции

### isLatexElement(line)
Проверяет, является ли строка inline LaTeX элементом.

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является inline LaTeX элементом

### isBoxLatexElement(line)
Проверяет, является ли строка блочным LaTeX элементом.

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является блочным LaTeX элементом

### isImageElement(line)
Проверяет, является ли строка изображением.

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является изображением

### isHandleElement(line)
Проверяет, является ли строка элементом-маркером (handle).

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является элементом-маркером

### isSymbolElement(line)
Проверяет, является ли строка символьным элементом.

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является символьным элементом

### isTagElement(line)
Проверяет, является ли строка тегом (element).

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является тегом

### isLinkElement(line)
Проверяет, является ли строка ссылкой.

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является ссылкой

### isBgColorElement(line)
Проверяет, является ли строка цветовым элементом фона.

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является цветовым элементом фона

### isColorElement(line)
Проверяет, является ли строка цветовым элементом.

**Параметры:**
- `line` (string): Строка для проверки

**Возвращает:**
- `boolean`: true если строка является цветовым элементом

### detectElementType(line)
Определяет тип элемента по строке, используя все вышеуказанные функции.

**Параметры:**
- `line` (string): Строка для анализа

**Возвращает:**
- `string|null`: Тип элемента или null если тип не определен

Возможные возвращаемые значения:
- 'latex' - inline LaTeX
- 'boxLatex' - блочный LaTeX
- 'image' - изображение
- 'handle' - элемент-маркер
- 'symbol' - символьный элемент
- 'element' - тег
- 'link' - ссылка
- 'bgColor' - цвет фона
- 'color' - цвет текста
- null - не определен
