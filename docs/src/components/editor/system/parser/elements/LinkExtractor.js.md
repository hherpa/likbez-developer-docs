# LinkExtractor.js

## Общее описание
Файл `LinkExtractor.js` содержит функции для извлечения ссылок на узлы из текстового контента.

## Функции

### extractLinkElements(content)
Извлекает ссылки на узлы из контента.

**Параметры:**
- `content` (string): Контент для анализа

**Возвращает:**
- `Array`: Массив ссылок

**Структура элементов:**
Каждый элемент в массиве имеет следующую структуру:
- `nodeId` (string): Идентификатор узла (без тегов)
- `content` (string): Содержимое ссылки (то же, что и nodeId)
- `fullMatch` (string): Полное совпадение с тегами
- `position` (number): Позиция элемента в исходном тексте

**Процесс извлечения:**
1. Проверяет входные данные
2. Извлекает ссылки по паттерну `link`
3. Возвращает массив ссылок

## Зависимости
- `PARSER_CONFIG` из `../core/ParserConfig.js` (используется паттерн для извлечения ссылок)

## Использование
Функция используется в процессе парсинга для извлечения ссылок на другие узлы из текста, которые затем могут быть отрендерены в редакторе.
