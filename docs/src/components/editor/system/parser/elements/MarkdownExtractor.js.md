# MarkdownExtractor.js

## Общее описание
Файл `MarkdownExtractor.js` содержит функции для извлечения заголовка из текстового контента. Заголовок извлекается только из тега {.name}, H1 заголовок не используется для имени узла.

## Функции

### extractTitle(content)
Извлекает заголовок из контента.

**Параметры:**
- `content` (string): Контент для анализа

**Возвращает:**
- `string`: Заголовок или пустая строка

**Процесс извлечения:**
1. Проверяет входные данные
2. Ищет только тег {.name} по паттерну `name`
3. Возвращает содержимое тега или пустую строку, если тег не найден

## Зависимости
- `PARSER_CONFIG` из `../core/ParserConfig.js` (используется паттерн для извлечения имени)

## Использование
Функция используется в процессе парсинга для извлечения заголовка узла из текста.
