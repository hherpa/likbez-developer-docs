# LatexExtractor.js

## Общее описание
Файл `LatexExtractor.js` содержит функции для извлечения LaTeX элементов из текстового контента. Поддерживаются как inline LaTeX формулы, так и блочные LaTeX элементы.

## Функции

### extractLatexElements(content)
Извлекает LaTeX элементы из контента.

**Параметры:**
- `content` (string): Контент для анализа

**Возвращает:**
- `Object`: Объект с массивами LaTeX элементов:
  - `latex` (Array): Массив inline LaTeX элементов
  - `boxLatex` (Array): Массив блочных LaTeX элементов

**Структура элементов:**
Каждый элемент в массивах имеет следующую структуру:
- `content` (string): Содержимое формулы (без тегов)
- `fullMatch` (string): Полное совпадение с тегами
- `position` (number): Позиция элемента в исходном тексте

**Процесс извлечения:**
1. Проверяет входные данные
2. Извлекает блочные LaTeX элементы по паттерну `boxLatex`
3. Извлекает inline LaTeX элементы по паттерну `latex`
4. Возвращает объект с массивами элементов

## Зависимости
- `PARSER_CONFIG` из `../core/ParserConfig.js` (используются паттерны для извлечения)

## Использование
Функция используется в процессе парсинга для извлечения математических формул из текста, которые затем могут быть отрендерены в редакторе.
