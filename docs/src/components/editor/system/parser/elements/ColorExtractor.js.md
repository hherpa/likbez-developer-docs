# ColorExtractor.js

## Общее описание
Файл `ColorExtractor.js` содержит функции для извлечения цветовых элементов из текстового контента. Поддерживаются как цвет текста, так и цвет фона.

## Функции

### extractColorElements(content)
Извлекает цветовые элементы из контента.

**Параметры:**
- `content` (string): Контент для анализа

**Возвращает:**
- `Array`: Массив цветовых элементов

**Структура элементов:**
Каждый элемент в массиве имеет следующую структуру:
- `color` (string): Название цвета
- `colorType` (string): Тип цвета (имя цвета или 'bg-имя цвета' для фона)
- `content` (string): Содержимое элемента (без тегов)
- `fullMatch` (string): Полное совпадение с тегами
- `type` (string): Тип элемента ('color')
- `position` (number): Позиция элемента в исходном тексте

**Процесс извлечения:**
1. Проверяет входные данные
2. Извлекает цветовые теги текста по паттерну `colorText`
3. Исключает LaTeX теги из цветового парсинга
4. Извлекает цветовые теги фона по паттерну `bgColor`
5. Возвращает массив цветовых элементов

## Зависимости
- `PARSER_CONFIG` из `../core/ParserConfig.js` (используются паттерны для извлечения цветов)

## Использование
Функция используется в процессе парсинга для извлечения цветовых элементов из текста, которые затем могут быть отрендерены в редакторе.
