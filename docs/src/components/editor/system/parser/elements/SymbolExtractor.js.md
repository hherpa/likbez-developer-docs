# SymbolExtractor.js

## Общее описание
Файл `SymbolExtractor.js` содержит функции для извлечения символьных элементов из текстового контента.

## Функции

### extractSymbolElements(content)
Извлекает символьные элементы из контента.

**Параметры:**
- `content` (string): Контент для анализа

**Возвращает:**
- `Array`: Массив символьных элементов

**Структура элементов:**
Каждый элемент в массиве имеет следующую структуру:
- `content` (string): Исходный текст символов (без тегов)
- `symbols` (Array): Массив отдельных символов
- `fullMatch` (string): Полное совпадение с тегами
- `position` (number): Позиция элемента в исходном тексте

**Процесс извлечения:**
1. Проверяет входные данные
2. Извлекает символьные элементы по паттерну `symbol`
3. Разбивает текст на отдельные символы по запятым и точкам с запятой
4. Создает один элемент для всей группы символов
5. Возвращает массив символьных элементов

## Зависимости
- `PARSER_CONFIG` из `../core/ParserConfig.js` (используется паттерн для извлечения символов)

## Использование
Функция используется в процессе парсинга для извлечения символьных элементов из текста, которые затем могут быть отрендерены в редакторе.
