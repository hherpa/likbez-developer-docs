# HandleExtractor.js

## Общее описание
Файл `HandleExtractor.js` содержит функции для извлечения элементов-маркеров (handle) из текстового контента.

## Функции

### extractHandleElements(content)
Извлекает элементы-маркеры (handle) из контента.

**Параметры:**
- `content` (string): Контент для анализа

**Возвращает:**
- `Array`: Массив элементов-маркеров

**Структура элементов:**
Каждый элемент в массиве имеет следующую структуру:
- `content` (string): Содержимое маркера (без тегов)
- `label` (string): Метка маркера (то же, что и content)
- `fullMatch` (string): Полное совпадение с тегами
- `position` (number): Позиция элемента в исходном тексте

**Процесс извлечения:**
1. Проверяет входные данные
2. Извлекает элементы-маркеры по паттерну `handle`
3. Возвращает массив элементов-маркеров

## Зависимости
- `PARSER_CONFIG` из `../core/ParserConfig.js` (используется паттерн для извлечения маркеров)

## Использование
Функция используется в процессе парсинга для извлечения элементов-маркеров из текста, которые затем могут быть отрендерены в редакторе.
