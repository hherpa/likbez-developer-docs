# MediaExtractor.js

## Общее описание
Файл `MediaExtractor.js` содержит функции для извлечения медиа элементов (изображений) из текстового контента.

## Функции

### extractMediaElements(content)
Извлекает медиа элементы (изображения) из контента.

**Параметры:**
- `content` (string): Контент для анализа

**Возвращает:**
- `Array`: Массив изображений

**Структура элементов:**
Каждый элемент в массиве имеет следующую структуру:
- `url` (string): URL изображения (без тегов)
- `fullMatch` (string): Полное совпадение с тегами
- `position` (number): Позиция элемента в исходном тексте

**Процесс извлечения:**
1. Проверяет входные данные
2. Извлекает изображения по паттерну `image`
3. Возвращает массив изображений

## Зависимости
- `PARSER_CONFIG` из `../core/ParserConfig.js` (используется паттерн для извлечения изображений)

## Использование
Функция используется в процессе парсинга для извлечения изображений из текста, которые затем могут быть отрендерены в редакторе.
