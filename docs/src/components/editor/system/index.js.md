# Документация для src/components/editor/system/index.js

## Описание

Файл src/components/editor/system/index.js является точкой экспорта для компонентов и утилит системы элементов редактора. Он предоставляет доступ к функциям создания и обновления узлов, а также экспортирует константы редактора.

## Экспортируемые функции

### createNode
Функция для создания нового узла с markdown контентом.

**Параметры:**
- position - Позиция узла
- options - Опции конфигурации узла (необязательный)

**Возвращает:** Объект узла с идентификатором, типом, позицией и данными

### updateNodeInList
Функция для обновления узла в списке узлов.

**Параметры:**
- nodes - Массив узлов
- nodeId - Идентификатор узла для обновления
- nodeData - Новые данные узла

**Возвращает:** Новый массив узлов с обновленным узлом

## Экспортируемые константы

Файл также экспортирует все константы из src/components/editor/constants/index.js, включая:

- Конфигурации редактора
- Настройки кнопок
- Шаблоны редактора
- Настройки выпадающих списков
- UI константы
- Палитру цветов

## Структура экспорта

```javascript
// Утилиты
export { createNode, updateNodeInList } from './parser/core/NodeFactory';

// Константы
export * from '../constants';
```

## Зависимости

- src/components/editor/system/parser/core/NodeFactory.js - содержит реализацию функций создания и обновления узлов
- src/components/editor/constants/index.js - точка экспорта всех констант редактора
