# LevelsTab Component Documentation

## Описание
LevelsTab - это компонент вкладки настроек уровней, который позволяет пользователю управлять уровнями узлов в приложении. Компонент предоставляет функциональность для добавления, удаления, редактирования и перетаскивания уровней для изменения их порядка.

## Экспортируемые компоненты
- `LevelsTab` - основной компонент вкладки уровней

## Структура компонента
1. **Заголовок вкладки**:
   - Заголовок "Уровни"
   - Описание функциональности
   - Кнопка добавления уровня

2. **Содержимое вкладки**:
   - Список уровней с возможностью редактирования
   - Поле ввода для редактирования названия уровня
   - Кнопки действий (редактирование, удаление)
   - Пустое состояние при отсутствии уровней

3. **Футер вкладки**:
   - Подсказка о возможности перетаскивания элементов

## Взаимодействия
- Использует хук `useSettings` для управления уровнями
- Состояние редактирования уровня (editingLevel, editName)
- Обработчики редактирования (начало, сохранение, отмена)
- Обработчики клавиш (Enter для сохранения, Escape для отмены)
- Поддержка drag-and-drop для изменения порядка уровней:
  - `handleDragStart` - начало перетаскивания
  - `handleDragOver` - обработка перетаскивания над элементом
  - `handleDrop` - завершение перетаскивания и изменение порядка

## Зависимости
- React
- `useSettings` из './SettingsProvider'
- `SettingsIcons` из './SettingsIcons'
- CSS стили из './style/LevelsTab.css'

## Архитектура
Компонент реализует паттерн управления списком с возможностью редактирования элементов. Использует локальное состояние для управления режимом редактирования и значением редактируемого поля. Реализована поддержка drag-and-drop для изменения порядка элементов списка. Компонент инкапсулирует всю логику управления уровнями, включая валидацию ввода и обработку пользовательских взаимодействий. Использует иконки из SettingsIcons для визуального оформления кнопок действий.
