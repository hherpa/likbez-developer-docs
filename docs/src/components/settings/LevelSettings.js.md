# LevelSettings Component Documentation

## Описание
LevelSettings - это компонент настроек, который отображает интерфейс для управления темой приложения и настройками уровней. Компонент предоставляет пользователю возможность переключать тему (светлая/темная) и управлять уровнями узлов.

## Экспортируемые компоненты
- `LevelSettings` - основной компонент настроек

## Структура компонента
1. **Тема секция**:
   - Заголовок "Тема"
   - Описание функциональности
   - Кнопка переключения темы (Светлая/Темная)

2. **Уровни секция**:
   - Заголовок "Уровни"
   - Описание функциональности
   - Кнопка управления уровнями

## Взаимодействия
- Использует хук `useSettings` для получения информации об уровнях
- Использует хук `useTheme` для управления темой
- При нажатии на кнопку переключения темы вызывается `toggleTheme`
- При нажатии на кнопку управления уровнями вызывается `onOpenLevelsTab` (получаемый через props)

## Зависимости
- React
- `useSettings` из './SettingsProvider'
- `useTheme` из '../../core/hooks/storage/useTheme'
- `SettingsIcons` из './SettingsIcons'
- CSS стили из './style/LevelSettings.css'

## Архитектура
Компонент следует паттерну функциональных компонентов React с использованием хуков для управления состоянием. Структура разделена на две секции (тема и уровни), каждая из которых имеет заголовок, описание и кнопку действия. Компонент принимает пропс `onOpenLevelsTab` для делегирования открытия вкладки уровней родительскому компоненту.
