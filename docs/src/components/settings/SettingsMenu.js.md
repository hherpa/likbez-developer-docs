# SettingsMenu Component Documentation

## Описание
SettingsMenu - это компонент меню настроек, который предоставляет интерфейс для управления различными настройками приложения. Компонент реализует систему вкладок с возможностью открытия/закрытия вкладок и переключения между ними.

## Экспортируемые компоненты
- `SettingsMenu` - основной компонент меню настроек

## Структура компонента
1. **Overlay**:
   - Затемненный фон, закрывающий основной контент
   - Обработчик закрытия при клике вне меню

2. **Header**:
   - Заголовок "Настройки" с иконкой
   - Кнопка закрытия меню

3. **Tabs**:
   - Список открытых вкладок
   - Кнопки переключения между вкладками
   - Кнопки закрытия вкладок (кроме основной)

4. **Content**:
   - Активная вкладка с содержимым

5. **Footer**:
   - Подсказка о автоматическом сохранении настроек

## Взаимодействия
- Управление состоянием открытых вкладок (openTabs)
- Управление активной вкладкой (activeTab)
- Обработчики открытия вкладки уровней (openLevelsTab)
- Обработчики закрытия вкладок (closeTab)
- Обновление ширины вкладок для корректного отображения (updateTabWidths)
- Обработчик закрытия меню (onClose)

## Зависимости
- React (useState, useRef, useEffect)
- `LevelSettings` из './LevelSettings'
- `LevelsTab` из './LevelsTab'
- `SettingsIcons` из './SettingsIcons'
- CSS стили из './style/SettingsMenu.css'

## Архитектура
Компонент реализует паттерн системы вкладок с динамическим управлением. Использует состояние для отслеживания открытых вкладок и активной вкладки. Реализована система динамического открытия/закрытия вкладок с ограничением на закрытие основной вкладки. Компонент использует refs для управления шириной вкладок и обеспечивает корректное отображение интерфейса. Взаимодействие с содержимым вкладок осуществляется через передачу пропсов, включая обработчик открытия вкладки уровней. Поддерживает темизацию через пропс darkMode.
