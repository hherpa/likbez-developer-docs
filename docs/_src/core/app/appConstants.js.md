# appConstants.js

## Обзор

Файл содержит константы приложения, которые используются в основном компоненте App.js и других частях приложения. Включает настройки темы и идентификаторы по умолчанию.

## Константы

### APP_THEME
Объект, содержащий настройки цветовой схемы для светлой и темной тем приложения.

#### Светлая тема (light)
- `background`: '#ffffff' - белый фон
- `text`: '#111111' - черный текст

#### Темная тема (dark)
- `background`: '#0f1115' - темно-серый фон
- `text`: '#e5e7eb' - светло-серый текст

!> Эти значения жестко закодированы и могут потребовать изменения при обновлении дизайна приложения.

### DEFAULT_LEVEL_ID
Строка, содержащая идентификатор уровня по умолчанию: 'level1'

!> Значение 'level1' жестко закодировано и используется как fallback в навигации.

## Использование

Константы импортируются в других файлах следующим образом:
```javascript
import { APP_THEME, DEFAULT_LEVEL_ID } from '../appConstants';
```

## Зависимости

Файл не имеет внешних зависимостей и экспортирует простые объекты и строки.

## Принципы организации

1. **Централизация** - все глобальные константы приложения в одном месте
2. **Простота** - только необходимые константы без сложной логики
3. **Масштабируемость** - легко добавлять новые константы по мере необходимости
