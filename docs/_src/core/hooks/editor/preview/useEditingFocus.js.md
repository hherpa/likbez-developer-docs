# useEditingFocus

Управление фокусом при редактировании.

## Описание

Отвечает ТОЛЬКО за установку фокуса (SRP).

## Параметры

- `isEditing` (boolean): Режим редактирования

## Возвращаемое значение

- `editRef`: ref для элемента редактирования

## Особенности

- Автоматически устанавливает фокус на элемент при начале редактирования
- Устанавливает курсор в конец текста
- Очищает выделение при установке фокуса

## Использование

```javascript
const editRef = useEditingFocus(isEditing);

// В JSX:
<div
  ref={editRef}
  contentEditable={isEditing}
>
  {content}
</div>
```
