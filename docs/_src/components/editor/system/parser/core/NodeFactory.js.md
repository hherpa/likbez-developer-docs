# NodeFactory.js

## Общее описание
Файл `NodeFactory.js` содержит утилиты для создания и управления узлами в редакторе. Основные функции включают создание шаблонов контента, создание новых узлов и обновление существующих узлов.

## Функции

### createTemplate(options)
Создает шаблон контента в markdown формате.

**Параметры:**
- `options` (Object): Опции для создания шаблона
  - `title` (string): Заголовок узла (по умолчанию 'Новый узел')
  - `tag` (string): Тег узла (по умолчанию пустая строка)
  - `text` (string): Текст описания (по умолчанию 'Описание узла...')
  - `symbols` (string): Символы узла (по умолчанию пустая строка)
  - `handles` (Array): Массив элементов-маркеров (по умолчанию пустой массив)
  - `images` (Array): Массив изображений (по умолчанию пустой массив)
  - `latex` (string): LaTeX формула (по умолчанию пустая строка)

**Возвращает:**
- `string`: Шаблон контента в markdown формате

### createNode(position, options)
Создает новый узел с markdown контентом.

**Параметры:**
- `position` (Object): Позиция узла на диаграмме
- `options` (Object): Опции для создания узла
  - Наследует все параметры от `createTemplate`
  - `type` (string): Тип узла
  - `data` (Object): Дополнительные данные узла

**Возвращает:**
- `Object`: Новый узел со следующей структурой:
  - `id` (string): Уникальный идентификатор узла
  - `type` (string): Тип узла
  - `position` (Object): Позиция узла
  - `data` (Object): Данные узла, включая контент и заголовок

### updateNodeInList(nodes, nodeId, nodeData)
Обновляет узел в списке узлов.

**Параметры:**
- `nodes` (Array): Массив узлов
- `nodeId` (string): Идентификатор узла для обновления
- `nodeData` (Object): Новые данные узла

**Возвращает:**
- `Array`: Обновленный массив узлов

## Зависимости
- `DEFAULT_NODE_CONFIG` из `../../../constants`

## Использование
Файл используется для создания и управления узлами в редакторе, обеспечивая стандартизированный способ генерации контента и структуры узлов.
