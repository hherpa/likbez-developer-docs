# ColorResolver.js

## Общее описание
Файл `ColorResolver.js` содержит утилиты для работы с цветами элементов в редакторе. Вместо создания отдельного класса используются простые функции для получения и проверки цветов элементов.

## Функции

### getElementColors(elementElements)
Получает цвета для element элементов.

**Параметры:**
- `elementElements` (Array): Массив element элементов

**Возвращает:**
- `Object`: Объект с цветами element элементов, где ключи - названия элементов, а значения - цвета

### getEmptyColors()
Получает пустой объект цветов.

**Возвращает:**
- `Object`: Пустой объект цветов

### getColor(colorName)
Получает цвет по имени из единого источника.

**Параметры:**
- `colorName` (string): Имя цвета

**Возвращает:**
- `string`: Цвет или цвет по умолчанию

### isSupportedColor(colorName)
Проверяет, поддерживается ли цвет.

**Параметры:**
- `colorName` (string): Имя цвета

**Возвращает:**
- `boolean`: Поддерживается ли цвет

## Зависимости
- `PARSER_CONFIG` из `./ParserConfig.js`
- `getColor` из `../../../constants/ColorPalette`

## Использование
Файл используется в процессе парсинга для определения цветов элементов и проверки их поддержки. Функции обеспечивают централизованное управление цветами в приложении.
