# ElementBuilder.js

## Описание
ElementBuilder - построение массива элементов. Отвечает только за создание массива элементов из результата парсинга.

## Экспортируемые компоненты
- `ElementBuilder` - построитель элементов

## Структура
- `buildElementArray(parsed)` - строит массив всех элементов из результата парсинга

## Взаимодействия
- Принимает результат парсинга и создает из него массив элементов
- Не имеет внешних зависимостей

## Зависимости
- Нет внешних зависимостей

## Архитектура
Построитель элементов реализует паттерн "Строитель" для создания массива элементов:
1. Метод `buildElementArray` принимает результат парсинга и создает из него массив элементов
2. Обрабатывает различные типы элементов:
   - Markdown элементы (включая LaTeX)
   - Изображения
   - Handle элементы
   - Символы
   - Element элементы
   - Ссылки
3. Исключает дублирующие элементы (cleanText)
4. Добавляет тип элемента для корректного рендеринга

Класс следует принципу единственной ответственности, отвечая только за создание массива элементов из результата парсинга. Не содержит логики рендеринга или сортировки элементов.

Архитектура позволяет легко расширять поддержку новых типов элементов, добавляя их обработку в метод `buildElementArray`.
