# ColorRenderer.js

## Описание
ColorRenderer - рендерер для цветного текста. Соблюдает SOLID принципы.

## Экспортируемые компоненты
- `ColorRenderer` - рендерер цветного текста
- `ColorRendererComponent` - функциональный компонент для обратной совместимости

## Структура
- `constructor()` - инициализация рендерера для типа 'color'
- `renderElement(element, context)` - рендерит цветной текст
- `getColorStyle(colorType)` - возвращает стили для цвета

## Взаимодействия
- Наследуется от BaseRenderer
- Использует React для рендеринга элементов

## Зависимости
- `react` - для рендеринга React элементов
- `../core/BaseRenderer` - базовый класс рендерера

## Архитектура
Рендерер цветного текста реализует паттерн "Стратегия" для различных типов цветового оформления:
1. Метод `renderElement` принимает элемент и контекст рендеринга
2. Поддерживает различные типы цветов (red, green, blue, yellow, purple, orange)
3. Метод `getColorStyle` возвращает стили в зависимости от типа цвета:
   - Обычный цвет - цветной текст
   - Цвет фона (bg-*) - текст с цветным фоном
4. Поддерживает темизацию через контекст

Функциональный компонент `ColorRendererComponent` обеспечивает обратную совместимость для использования в JSX.

Класс следует принципу единственной ответственности, отвечая только за рендеринг цветного текста. Поддерживает как цветной текст, так и текст с цветным фоном.
