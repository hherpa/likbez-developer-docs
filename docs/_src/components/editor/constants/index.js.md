# constants/index.js

## Общее описание
Файл `index.js` в директории `constants` является точкой экспорта всех констант редактора. Он объединяет и экспортирует различные группы констант для удобного использования в других частях приложения.

## Экспорты

### Конфигурации
Экспортирует все константы из следующих файлов:
- `./EditorConfig` - основные конфигурации редактора
- `./EditorButtons` - конфигурации кнопок редактора
- `./EditorTemplates` - шаблоны редактора
- `./EditorDropdowns` - конфигурации выпадающих списков
- `./EditorUI` - конфигурации пользовательского интерфейса
- `./EditorSettings` - настройки редактора

### Цвета
Экспортирует все константы из:
- `./ColorPalette` - цветовая палитра редактора

## Использование
Файл используется для централизованного импорта всех констант редактора. Другие модули могут импортировать нужные константы напрямую из этой директории.
