# EditorConfig.js

## Общее описание
Файл `EditorConfig.js` содержит общие конфигурации редактора, включая системные константы и конфигурации узлов.

## Константы

### LOADING_INDICATOR_CONFIG
Конфигурация индикатора загрузки:
- `position` (string): 'fixed' - фиксированное позиционирование
- `top` (string): '20px' - отступ сверху
- `right` (string): '20px' - отступ справа
- `borderRadius` (string): '8px' - радиус скругления углов
- `padding` (string): '12px 16px' - внутренние отступы
- `zIndex` (number): 1001 - порядок наложения
- `display` (string): 'flex' - тип отображения
- `alignItems` (string): 'center' - выравнивание по центру
- `gap` (string): '8px' - расстояние между элементами

### SPINNER_CONFIG
Конфигурация спиннера:
- `width` (string): '16px' - ширина
- `height` (string): '16px' - высота
- `borderRadius` (string): '50%' - круглая форма
- `animation` (string): 'spin 1s linear infinite' - анимация вращения

### DEFAULT_NODE_CONFIG
Дефолтные значения для создания узлов:
- `title` (string): 'Новый узел' - заголовок по умолчанию
- `text` (string): 'Описание узла...' - текст по умолчанию
- `tag` (string): '' - тег по умолчанию (пустой)
- `symbols` (string): '' - символы по умолчанию (пустые)
- `latex` (string): '' - LaTeX формула по умолчанию (пустая)
- `images` (Array): [] - массив изображений по умолчанию (пустой)
- `type` (string): 'base' - тип узла по умолчанию

## Использование
Константы используются для настройки внешнего вида и поведения различных элементов редактора, а также для задания значений по умолчанию при создании новых узлов.
