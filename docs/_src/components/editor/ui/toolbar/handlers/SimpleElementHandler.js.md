# SimpleElementHandler.js

## Назначение
Обработчик простых элементов (заголовков и параграфов) в редакторе.

## Экспортируемые компоненты
- `handleHeaderElement` - функция обработки заголовков
- `handleParagraphElement` - функция обработки параграфов

## Структура
1. Константы для префиксов заголовков
2. Вспомогательная функция `getHeaderPrefix`
3. Функция `handleHeaderElement` для обработки заголовков
4. Функция `handleParagraphElement` для обработки параграфов

## Взаимодействия
1. Получает тип элемента, контент и элементы редактирования
2. Для заголовков: получает текст без префикса, добавляет префикс заголовка, вставляет элемент
3. Для параграфов: разбивает текст на строки, вставляет элемент
4. Обновляет массив строк

## Зависимости
- `ElementInserter.js` - для построения текста вставки
- `ContentEditableUtils.js` - для получения текста из contentEditable
- `Logger.js` - для логирования

## Архитектура
Следует принципу единственной ответственности (SRP), отвечая только за обработку заголовков и параграфов. Объединяет HeaderElementHandler и ParagraphElementHandler для упрощения структуры.

Функция `handleHeaderElement` реализует логику:
1. Получает текст из contentEditable
2. Удаляет префиксы заголовков из текста
3. Получает префикс для типа заголовка
4. Строит текст для вставки
5. Обновляет целевую строку с префиксом
6. Вставляет элемент после целевой строки

Функция `handleParagraphElement` реализует логику:
1. Получает текст из contentEditable
2. Разбивает текст на строки без пустых строк
3. Строит текст для вставки
4. Если есть строки, заменяет целевую строку на строки текста и вставляет элемент
5. Если нет строк, заменяет целевую строку на элемент
