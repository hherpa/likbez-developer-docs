# index.css

## Описание
Главный файл стилей редактора, который импортирует все остальные CSS файлы и определяет CSS переменные для темизации редактора.

## Структура
- Импорт стилей:
  - `./Editor.css` - основные стили редактора
  - `./Toolbar.css` - стили тулбара
  - `./Tabs.css` - стили вкладок
  - `./Textarea.css` - стили текстовой области
  - `./Preview.css` - стили предпросмотра
- Глобальные стили для textarea (отключение outline, box-shadow, border при фокусе)
- CSS переменные для светлой темы (:root)
- CSS переменные для темной темы (.dark-theme)

## Взаимодействия
- Импортирует все основные CSS файлы компонентов редактора
- Определяет глобальные CSS переменные, используемые во всех компонентах

## Зависимости
- `./Editor.css` - основные стили редактора
- `./Toolbar.css` - стили тулбара
- `./Tabs.css` - стили вкладок
- `./Textarea.css` - стили текстовой области
- `./Preview.css` - стили предпросмотра

## Архитектура
Файл служит точкой входа для всех стилей редактора. Он:
1. Импортирует все отдельные CSS файлы компонентов
2. Определяет CSS переменные для темизации (светлая и темная темы)
3. Устанавливает глобальные стили для textarea

CSS переменные разделены на категории:
- Editor Container
- Toolbar
- Tabs
- Textarea
- Line Numbers
- Preview
- Info Panel

Каждая тема (светлая и темная) имеет свои значения для всех переменных, что позволяет легко переключаться между темами.
