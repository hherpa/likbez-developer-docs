# LineProcessor.js

## Назначение
Модуль для обработки строк контента в редакторе, отвечающий за поиск строк по позиции в контенте.

## Экспортируемые компоненты
- `findLineByPosition` - функция поиска строки по позиции

## Структура
1. Функция `findLineByPosition` для поиска строки по позиции

## Взаимодействия
1. Принимает массив строк и позицию в контенте
2. Вычисляет позиции строк с учетом переносов
3. Сравнивает позицию с границами строк
4. Возвращает индекс найденной строки или -1

## Зависимости
- Нет внешних зависимостей

## Архитектура
Следует принципу единственной ответственности (SRP), отвечая только за обработку строк.

Функция `findLineByPosition` реализует логику:
1. Инициализирует текущую позицию в 0
2. Проходит по всем строкам:
   - Вычисляет начало и конец строки
   - Сравнивает позицию с границами строки
   - Если позиция находится в строке, возвращает индекс
   - Увеличивает текущую позицию на длину строки + 1 (перенос)
3. Если строка не найдена, возвращает -1
