# TextFinder.js

## Назначение
Модуль для поиска текста в строке в редакторе, отвечающий за поиск позиции текста и удаление текста из строки.

## Экспортируемые компоненты
- `findTextInLine` - функция поиска позиции текста в строке
- `removeTextFromLine` - функция удаления текста из строки

## Структура
1. Импорт логгера
2. Функция `findTextInLine` для поиска текста в строке
3. Функция `removeTextFromLine` для удаления текста из строки

## Взаимодействия
1. Использует логгер для отладки процесса поиска
2. Работает со строками для поиска и удаления текста
3. Использует регулярные выражения для сложных стратегий поиска
4. Принимает строку и текст для поиска/удаления
5. Возвращает позицию и длину найденного текста или новую строку

## Зависимости
- `Logger.js` - для логирования

## Архитектура
Следует принципу единственной ответственности (SRP), отвечая только за поиск текста.

Функция `findTextInLine` реализует логику:
1. Проверяет валидность входных параметров
2. Удаляет пробелы по краям у искомого текста
3. Логирует процесс поиска
4. Применяет несколько стратегий поиска:
   - Точное совпадение
   - Поиск без учета регистра
   - Поиск с учетом пробелов вокруг
   - Поиск по словам
   - Поиск одного слова как части текста
5. Возвращает позицию и длину найденного текста или null

Функция `removeTextFromLine` реализует логику:
1. Разбивает строку на части до и после удаляемого текста
2. Удаляет пробелы по краям у частей
3. Объединяет части с пробелом между ними
4. Обрабатывает случаи, когда одна из частей пустая
5. Возвращает строку без удаленного текста
