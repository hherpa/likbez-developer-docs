# Документация Likbez

Добро пожаловать в документацию приложения Likbez!

## О проекте

Likbez - это приложение для создания и управления интерактивными диаграммами и документацией.

## Структура проекта

- `src/` - исходный код приложения
- `electron/` - код для Electron версии
- `public/` - статические файлы
- `docs/` - документация (вы здесь)

## Архитектура

Приложение построено на следующих технологиях:

- React
- React Flow
- Electron (для десктопной версии)
- Docsify (для документации)

## Компоненты

### Диаграммы

Компоненты для работы с диаграммами находятся в `src/components/diagram/`.

### Редактор

Редактор документов находится в `src/components/editor/`.

### Настройки

Компоненты настроек находятся в `src/components/settings/`.

### Боковая панель

Компоненты боковой панели находятся в `src/components/sidebar/`.

## Разработка

### Компоненты

Все компоненты React находятся в директории `src/components/`.

### Хуки

Пользовательские хуки находятся в `src/core/hooks/`.

### Провайдеры

Контекстные провайдеры находятся в `src/core/providers/`.
