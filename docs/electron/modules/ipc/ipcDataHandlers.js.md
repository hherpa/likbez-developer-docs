# modules/ipc/ipcDataHandlers.js

## Описание

Обработчики IPC сообщений для работы с данными в Electron приложении Likbez. Регистрирует обработчики для инициализации, чтения, записи данных и получения путей к директориям узлов.

## Основные функции

### registerDataHandlers(ipcMain, dataManager)

Регистрирует обработчики IPC сообщений для работы с данными:

### Обработчик 'data:init'

Инициализирует структуру данных:

- Вызывает метод initializeData у dataManager
- Возвращает результат инициализации (true/false)
- Обрабатывает ошибки с логированием

### Обработчик 'data:read'

Читает данные из хранилища:

- Вызывает метод loadFromDisk у dataManager
- Возвращает загруженные данные (subLevels, categoryTags, levelsConfig)
- Обрабатывает ошибки с логированием и возвратом пустых данных

### Обработчик 'data:write'

Записывает данные в хранилище:

- Вызывает метод serializeToDisk у dataManager
- Возвращает результат записи (true/false)
- Обрабатывает ошибки с логированием

### Обработчик 'data:getNodeDirectory'

Получает путь к директории узла:

- Вызывает метод getNodeDirectory у dataManager
- Возвращает путь к директории узла или null
- Обрабатывает ошибки с логированием

## Параметры функции

- ipcMain: объект ipcMain из Electron для регистрации обработчиков
- dataManager: экземпляр DataManager для выполнения операций с данными

## Обрабатываемые сообщения

- data:init - инициализация структуры данных
- data:read - чтение данных из хранилища
- data:write - запись данных в хранилище
- data:getNodeDirectory - получение пути к директории узла

## Импорты

Модуль не содержит импортов, так как получает все необходимые зависимости через параметры функции registerDataHandlers.
