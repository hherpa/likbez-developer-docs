# modules/ipc/ipcWatcherHandlers.js

## Описание

Обработчики IPC сообщений для работы с отслеживанием изменений файловой системы в Electron приложении Likbez. Регистрирует обработчики для запуска и остановки watcher'а.

## Основные функции

### registerWatcherHandlers(ipcMain, dataManager)

Регистрирует обработчики IPC сообщений для работы с watcher'ом:

### Обработчик 'watcher:start'

Запускает отслеживание изменений файловой системы:

- Вызывает метод startWatching у dataManager
- Устанавливает callback для отправки уведомлений об изменениях
- Возвращает результат запуска (true/false)
- Обрабатывает ошибки с логированием

### Обработчик 'watcher:stop'

Останавливает отслеживание изменений файловой системы:

- Вызывает метод stopWatching у dataManager
- Возвращает результат остановки (true/false)
- Обрабатывает ошибки с логированием

## Параметры функции

- ipcMain: объект ipcMain из Electron для регистрации обработчиков
- dataManager: экземпляр DataManager для управления watcher'ом

## Обрабатываемые сообщения

- watcher:start - запуск отслеживания изменений
- watcher:stop - остановка отслеживания изменений

## Отправляемые сообщения

- watcher:change - уведомление об изменении файловой системы (eventType, filename)

## Импорты

Модуль не содержит импортов, так как получает все необходимые зависимости через параметры функции registerWatcherHandlers.
