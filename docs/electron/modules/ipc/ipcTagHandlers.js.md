# modules/ipc/ipcTagHandlers.js

## Описание

Обработчики IPC сообщений для работы с тегами в Electron приложении Likbez. Регистрирует обработчики для чтения и записи тегов категорий.

## Основные функции

### registerTagHandlers(ipcMain)

Регистрирует обработчики IPC сообщений для работы с тегами:

### Обработчик 'tags:read'

Читает теги категорий из файла:

- Получает путь к корневой директории данных
- Читает файл category_tags.json
- Возвращает теги или пустой объект при ошибке
- Обрабатывает ошибки с логированием

### Обработчик 'tags:write'

Записывает теги категорий в файл:

- Получает путь к корневой директории данных
- Записывает теги в файл category_tags.json
- Возвращает результат записи (true/false)
- Обрабатывает ошибки с логированием

## Параметры функции

- ipcMain: объект ipcMain из Electron для регистрации обработчиков

## Обрабатываемые сообщения

- tags:read - чтение тегов категорий
- tags:write - запись тегов категорий

## Импорты

- path: модуль для работы с путями файловой системы
- JsonUtils из ../utils: утилиты для работы с JSON
- FileUtils из ../utils: утилиты для работы с файловой системой
