# ColorPalette.js

## 1. Назначение файла

Файл `src/components/editor/constants/ColorPalette.js` является централизованным источником цветовой палитры для всего приложения. Он обеспечивает единообразное использование цветов во всех компонентах системы, включая парсер, рендерер, пользовательский интерфейс и выпадающие списки.

## 2. Экспортируемые константы и их структура

### COLORS
Объект, содержащий палитру предопределенных цветов:
- `red`: '#e57373' (приятный красный)
- `green`: '#81c784' (приятный зеленый)
- `blue`: '#64b5f6' (приятный синий)
- `yellow`: '#ffd54f' (приятный желтый)
- `orange`: '#ffb74d' (приятный оранжевый)
- `purple`: '#ba68c8' (приятный фиолетовый)
- `pink`: '#f06292' (приятный розовый)
- `brown`: '#a1887f' (приятный коричневый)

### getColor(colorName)
Функция для получения значения цвета по его имени:
- Принимает имя цвета (string)
- Возвращает соответствующее значение цвета из палитры
- Если цвет не найден, возвращает черный цвет (#000000) по умолчанию

### SUPPORTED_COLORS
Массив, содержащий список всех поддерживаемых цветов (ключи объекта COLORS).

## 3. Использование цветовой палитры в приложении

Цветовая палитра используется как единый источник истины для всех цветовых значений в приложении. Это обеспечивает консистентность визуального оформления и упрощает изменение цветовой схемы во всем приложении через изменение одного файла.

## 4. Взаимодействие с другими компонентами

Файл используется в различных частях приложения:
- В парсере для обработки цветовых маркеров
- В рендерере для отображения цветных элементов
- В UI компонентах для стилизации интерфейса
- В выпадающих списках для отображения доступных цветов

## 5. Используемые зависимости

Файл не имеет внешних зависимостей и использует только встроенные возможности JavaScript (Object.keys).
