# TagExtractor.js

## Общее описание
Файл `TagExtractor.js` содержит функции для извлечения тегов (бывших element элементов) из текстового контента.

## Функции

### extractTagElements(content)
Извлекает теги (бывшие element элементы) из контента.

**Параметры:**
- `content` (string): Контент для анализа

**Возвращает:**
- `Array`: Массив тегов

**Структура элементов:**
Каждый элемент в массиве имеет следующую структуру:
- `element` (string): Название тега (без тегов)
- `content` (string): Содержимое тега (то же, что и element)
- `fullMatch` (string): Полное совпадение с тегами
- `position` (number): Позиция элемента в исходном тексте

**Процесс извлечения:**
1. Проверяет входные данные
2. Извлекает теги по паттерну `element`
3. Возвращает массив тегов

## Зависимости
- `PARSER_CONFIG` из `../core/ParserConfig.js` (используется паттерн для извлечения тегов)

## Использование
Функция используется в процессе парсинга для извлечения тегов из текста, которые затем могут быть отрендерены в редакторе.
