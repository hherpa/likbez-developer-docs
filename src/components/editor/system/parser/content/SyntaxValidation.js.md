# SyntaxValidation.js

## Общее описание
Файл `SyntaxValidation.js` содержит функции для валидации синтаксиса элементов в тексте редактора. Основная функция `validateSyntax` проверяет парность тегов и корректность специальных элементов, а вспомогательная функция `hasTags` проверяет наличие специальных тегов в контенте.

## Функции

### validateSyntax(content)
Валидирует синтаксис элементов в переданном контенте.

**Параметры:**
- `content` (string): Контент для валидации

**Возвращает:**
- `object`: Результат валидации с полями:
  - `valid` (boolean): true если валидация пройдена успешно
  - `errors` (Array): Массив строк с ошибками валидации
  - `warnings` (Array): Массив строк с предупреждениями

**Процесс валидации:**
1. Проверяет парность всех типов тегов из `PARSER_CONFIG.patterns`
2. Проверяет корректность URL изображений
3. Возвращает результат с ошибками и предупреждениями

### hasTags(content)
Проверяет, содержит ли контент специальные теги.

**Параметры:**
- `content` (string): Контент для проверки

**Возвращает:**
- `boolean`: true если контент содержит специальные теги

## Зависимости
- `src/components/editor/system/parser/core/ParserConfig.js` - Использует паттерны и настройки для валидации синтаксиса

## Использование
Файл используется в системе парсинга редактора для проверки корректности синтаксиса перед обработкой контента.
