# MarkdownHeadingRenderers.js

## Описание
MarkdownHeadingRenderers - рендереры заголовков в markdown. Обрабатывает рендеринг заголовков h1, h2, h3.

## Экспортируемые компоненты
- `renderH1(content, index, darkMode)` - функция рендеринга заголовка h1
- `renderH2(content, index, darkMode)` - функция рендеринга заголовка h2
- `renderH3(content, index, darkMode)` - функция рендеринга заголовка h3

## Структура
- `renderH1(content, index, darkMode)` - рендеринг заголовка h1:
  - Устанавливает размер шрифта 20px, жирность 700
  - Устанавливает отступы и цвет в зависимости от темы
- `renderH2(content, index, darkMode)` - рендеринг заголовка h2:
  - Устанавливает размер шрифта 18px, жирность 600
  - Устанавливает отступы и цвет в зависимости от темы
- `renderH3(content, index, darkMode)` - рендеринг заголовка h3:
  - Устанавливает размер шрифта 16px, жирность 600
  - Устанавливает отступы и цвет в зависимости от темы

## Взаимодействия
- Использует React для рендеринга элементов

## Зависимости
- `react` - для рендеринга React элементов

## Архитектура
Рендереры заголовков реализуют простой алгоритм рендеринга с учетом темы:
1. Каждая функция принимает содержимое заголовка, индекс и флаг темного режима
2. Возвращает соответствующий HTML элемент заголовка (h1, h2, h3)
3. Устанавливает стили в зависимости от типа заголовка и темы:
   - Размер шрифта (20px, 18px, 16px)
   - Жирность (700, 600, 600)
   - Цвет текста в зависимости от темы
   - Перенос слов и отступы

Функции следуют принципу единственной ответственности, отвечая только за рендеринг конкретного типа заголовка. Используются в основном рендерере markdown для отображения заголовков.
