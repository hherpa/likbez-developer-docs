# MarkdownInlineParser.js

## Описание
MarkdownInlineParser - парсер и рендерер inline markdown элементов. Обрабатывает форматирование внутри параграфов.

## Экспортируемые компоненты
- `renderInlineMarkdown(text, darkMode)` - функция рендеринга inline markdown

## Структура
- `renderInlineMarkdown(text, darkMode)` - основная функция рендеринга:
  - Парсит текст на составные части
  - Определяет типы элементов по паттернам
  - Фильтрует пересекающиеся элементы
  - Рендерит каждую часть с соответствующим стилем

## Взаимодействия
- Использует палитру цветов для цветового форматирования
- Использует InlineLatexRenderer для рендеринга LaTeX формул
- Использует React для рендеринга элементов

## Зависимости
- `react` - для рендеринга React элементов
- `../../../constants/ColorPalette` - палитра цветов
- `./InlineLatexRenderer` - рендерер inline LaTeX

## Архитектура
Парсер inline markdown реализует алгоритм поэтапной обработки текста:
1. Определяет паттерны для различных типов форматирования:
   - LaTeX формулы ({.LaTeX}...{.LaTeX})
   - Цветной фон ({.bg-color}...{.bg-color})
   - Цветной текст ({.color}...{.color})
   - Жирный текст (**...**)
   - Курсив (*...*)
   - Код (`...`)
   - Ссылки ([text](url))
2. Находит все совпадения в тексте
3. Фильтрует пересекающиеся элементы (оставляет первые по порядку)
4. Разбивает текст на части и рендерит каждую с соответствующим стилем:
   - bold - жирный текст
   - italic - курсив
   - code - моноширинный текст с фоном
   - link - ссылка с подчеркиванием
   - color - цветной текст
   - bgcolor - текст с цветным фоном
   - latex - LaTeX формула через InlineLatexRenderer
   - text - обычный текст

Функция следует принципу единственной ответственности, отвечая только за парсинг и рендеринг inline элементов. Использует делегирование для рендеринга LaTeX формул.
