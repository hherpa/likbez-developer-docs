# LinkRenderer.js

## Описание
LinkRenderer - рендерер для ссылок. Соблюдает SOLID принципы.

## Экспортируемые компоненты
- `LinkRenderer` - класс рендерера для ссылок
- `LinkRendererComponent` - функциональный компонент для обратной совместимости

## Структура
- Класс `LinkRenderer`, наследующий `BaseRenderer`
- Метод `renderElement` для рендеринга ссылок
- Функциональный компонент `LinkRendererComponent` для обратной совместимости

## Взаимодействие
- Наследует от `BaseRenderer` и реализует метод `renderElement`
- Использует `getThemeColors` для получения цветов темы
- Принимает элемент и контекст для рендеринга

## Зависимости
- `react` для создания компонентов
- `BaseRenderer` из '../core/BaseRenderer'

## Архитектура
Класс `LinkRenderer` наследует от `BaseRenderer` и реализует метод `renderElement` для рендеринга ссылок. Компонент создает span элемент с определенными стилями для отображения ссылки, включая цвет, подчеркивание, курсор и жирность шрифта. Также экспортируется функциональный компонент `LinkRendererComponent` для обратной совместимости, который создает экземпляр `LinkRenderer` и вызывает его метод `renderElement`.
