# SelectionUtils.js

## Назначение
Вспомогательные функции для работы с выделением текста в редакторе.

## Экспортируемые компоненты
- `getSelectedTextFromSelection` - функция получения выделенного текста

## Структура
1. Функция `getSelectedTextFromSelection` для получения выделенного текста

## Взаимодействия
1. Принимает контент как параметр
2. Проверяет наличие контента
3. Получает выделение из window
4. Проверяет наличие range в выделении
5. Проверяет, что range не collapsed
6. Возвращает текст из range или пустую строку

## Зависимости
- Нет внешних зависимостей

## Архитектура
Следует принципу единственной ответственности (SRP), предоставляя только вспомогательные функции для работы с выделением текста.

Функция `getSelectedTextFromSelection` реализует логику:
1. Проверяет наличие контента и возвращает его, если есть
2. Получает выделение из window (с проверкой на undefined)
3. Проверяет наличие range в выделении
4. Проверяет, что range не collapsed
5. Возвращает текст из range
6. Возвращает пустую строку, если нет выделения или range collapsed
