# ContentEditableUtils.js

## Назначение
Утилиты для работы с contentEditable элементами в редакторе, предоставляющие единый интерфейс для получения текста из contentEditable элементов.

## Экспортируемые компоненты
- `getContentEditableText` - функция получения текста из contentEditable элемента

## Структура
1. Функция `getContentEditableText` для получения текста из contentEditable элемента

## Взаимодействия
1. Принимает HTMLElement
2. Возвращает текст элемента или пустую строку
3. Использует innerText, если доступен, иначе textContent

## Зависимости
- Нет внешних зависимостей

## Архитектура
Следует принципу единственной ответственности (SRP), отвечая только за работу с contentEditable элементами. Предоставляет единую точку получения текста из contentEditable элементов.

Функция `getContentEditableText` реализует логику:
1. Проверяет наличие элемента
2. Возвращает innerText, если доступен
3. Возвращает textContent, если innerText недоступен
4. Возвращает пустую строку, если элемент отсутствует
