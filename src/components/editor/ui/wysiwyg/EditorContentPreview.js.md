# EditorContentPreview Component Documentation

## Описание
EditorContentPreview - это компонент предварительного просмотра контента редактора. Он отображает отрендеренный контент без использования React Flow, поддерживает редактирование элементов по двойному клику и обеспечивает предварительный просмотр контента в редакторе.

## Экспортируемые компоненты
- `EditorContentPreview` - основной компонент предварительного просмотра контента

## Структура компонента
1. **Контейнер предварительного просмотра**:
   - CSS класс "editor-content-preview"
   - Стили для адаптивного отображения
   - Поддержка темной/светлой темы

2. **Содержимое**:
   - Обработчик пустого контента
   - Обработчик ошибок рендеринга
   - Редактируемый или статический рендеринг контента

## Взаимодействия
- Принимает пропсы:
  - `content` - содержимое для отображения
  - `darkMode` - флаг темного режима
  - `onChange` - обработчик изменений (опциональный)
- Использует систему рендеринга из `../../system/renders`
- Использует `contentRenderer` для рендеринга контента
- Использует `EditableContentRenderer` для редактируемого контента
- Использует `parseMarkdownContent` для парсинга контента
- Использует `RenderContext` для контекста рендеринга
- Логирование через утилиту `Logger`

## Зависимости
- React (useMemo)
- Система рендеринга из '../../system/renders'
- `contentRenderer` из '../../system/renders/content/ContentRenderer'
- `RenderContext` из '../../system/renders/core/RenderContext'
- `parseMarkdownContent` из '../../system/parser'
- `EditableContentRenderer` из './editable/EditableContentRenderer'
- `log` из './utils/Logger'

## Архитектура
Компонент реализует паттерн функционального компонента React с использованием хуков для мемоизации вычислений. Компонент инкапсулирует всю логику рендеринга контента, включая обработку пустого контента и ошибок. Использует систему рендеринга для отображения контента с поддержкой различных элементов. Реализована поддержка редактируемого режима через `EditableContentRenderer` при наличии обработчика `onChange`. Компонент следует принципам композиции React, принимая контент и настройки через пропсы. Использует мемоизацию на уровне компонента для оптимизации производительности. Поддерживает темизацию через пропс `darkMode`.
