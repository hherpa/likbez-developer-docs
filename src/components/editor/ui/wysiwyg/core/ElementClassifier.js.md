# ElementClassifier.js

## Назначение
Модуль для классификации типов элементов и маппинга типов в редакторе, обеспечивающий определение типа элемента и преобразование типов.

## Экспортируемые компоненты
- `isColorElement` - проверка, является ли элемент цветовым
- `isBgColorElement` - проверка, является ли элемент фоновым цветом
- `isAnyColorElement` - проверка, является ли элемент цветовым (цвет или фон)
- `isHeaderElement` - проверка, является ли элемент заголовком
- `isParagraphElement` - проверка, является ли элемент параграфом
- `isInlineLatexElement` - проверка, является ли элемент inline LaTeX
- `isBlockElement` - проверка, является ли элемент блок-элементом
- `getElementTag` - получение тега элемента по его типу
- `createElementPattern` - создание регулярного выражения для поиска элемента

## Структура
1. Импорты констант
2. Функции классификации элементов:
   - `isColorElement` - для цветовых элементов
   - `isBgColorElement` - для элементов фонового цвета
   - `isAnyColorElement` - для всех цветовых элементов
   - `isHeaderElement` - для заголовков
   - `isParagraphElement` - для параграфов
   - `isInlineLatexElement` - для inline LaTeX
   - `isBlockElement` - для блок-элементов
3. Маппинг типов элементов на теги
4. Функция `getElementTag` для получения тега элемента
5. Функция `createElementPattern` для создания паттерна поиска элемента

## Взаимодействия
1. Использует SUPPORTED_COLORS из констант для проверки цветовых элементов
2. Принимает тип элемента и возвращает boolean для классификации
3. Принимает тип элемента и возвращает тег для маппинга
4. Принимает тип элемента и возвращает регулярное выражение для поиска

## Зависимости
- `../../../constants` - для получения поддерживаемых цветов

## Архитектура
Следует принципу единственной ответственности (SRP), отвечая только за определение типа элемента и маппинг. Объединен с ElementTypeMapper для устранения дублирования.

Функции классификации реализуют логику:
- `isColorElement` проверяет, содержится ли тип элемента в SUPPORTED_COLORS
- `isBgColorElement` проверяет, начинается ли тип с 'bg-' и содержится ли цвет в SUPPORTED_COLORS
- `isAnyColorElement` объединяет проверки цвета и фона
- `isHeaderElement` проверяет, является ли тип заголовком (h1, h2, h3)
- `isParagraphElement` проверяет, является ли тип параграфом (p)
- `isInlineLatexElement` проверяет, является ли тип inline LaTeX (latex)
- `isBlockElement` проверяет, что элемент не является цветовым, заголовком, параграфом или inline LaTeX

Функция `getElementTag` реализует логику:
1. Нормализует тип элемента (boxLatex -> box-latex)
2. Возвращает тег из маппинга или оригинальный тип

Функция `createElementPattern` реализует логику:
1. Получает тег элемента
2. Экранирует специальные символы для regex
3. Создает и возвращает регулярное выражение для поиска элемента
