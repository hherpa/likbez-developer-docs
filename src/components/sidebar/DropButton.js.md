# DropButton Component Documentation

## Описание
DropButton - это компонент кнопки перетаскивания, который позволяет пользователю создавать новые узлы в диаграмме путем перетаскивания элемента. Компонент поддерживает настройку типа узла и дополнительных данных.

## Экспортируемые компоненты
- `DropButton` - основной компонент кнопки перетаскивания

## Структура компонента
1. **Контейнер**:
   - CSS класс "drop-button"
   - Атрибут draggable для поддержки перетаскивания
   - Стили для размеров и отступов
   - Обработчик onDragStart

2. **Содержимое**:
   - Текстовая метка (по умолчанию "Перетащи, чтобы создать")

## Взаимодействия
- Поддерживает перетаскивание с помощью HTML5 Drag and Drop API
- При начале перетаскивания:
  - Устанавливает данные перетаскивания в формате JSON с типом узла и полезной нагрузкой
  - Устанавливает эффект перетаскивания как "move"
  - Вызывает опциональный обработчик onDragStart
- Принимает пропсы:
  - `nodeType` - тип создаваемого узла (по умолчанию 'infoex')
  - `label` - текстовая метка кнопки
  - `payload` - дополнительные данные для узла
  - `onDragStart` - обработчик начала перетаскивания

## Зависимости
- React
- CSS стили из './style/Buttons.css'

## Архитектура
Компонент реализует паттерн функционального компонента React с поддержкой HTML5 Drag and Drop API. Компонент полностью настраиваемый через пропсы, позволяя определять тип создаваемого узла, текстовую метку и дополнительные данные. Стили компонента определены inline для обеспечения фиксированных размеров. Компонент следует принципам композиции React, предоставляя гибкий интерфейс для создания различных типов перетаскиваемых элементов.
