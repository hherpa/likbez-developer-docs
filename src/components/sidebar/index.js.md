# Sidebar Index Documentation

## Описание
Файл index.js в директории sidebar служит точкой экспорта компонентов боковой панели. Он объединяет и экспортирует все основные компоненты боковой панели для удобного импорта в других частях приложения.

## Экспортируемые компоненты
- `DropButton` - компонент кнопки перетаскивания для создания узлов
- `Sidebar` - основной компонент боковой панели
- `BackButton` - компонент кнопки возврата
- `FileExplorer` - компонент файлового проводника

## Структура файла
Файл содержит только экспортные выражения для компонентов боковой панели без дополнительной логики.

## Взаимодействия
Файл не содержит логики взаимодействия, только объявления экспортов компонентов.

## Зависимости
- `DropButton` из './DropButton'
- `Sidebar` из './Sidebar'
- `BackButton` из './BackButton'
- `FileExplorer` из './FileExplorer'

## Архитектура
Файл следует паттерну barrel export, который позволяет упростить импорт компонентов боковой панели в других частях приложения. Вместо импорта каждого компонента по отдельности, можно импортировать все необходимые компоненты из одной точки. Это улучшает организацию кода и упрощает управление зависимостями между модулями.
