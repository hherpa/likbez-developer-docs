# BackButton Component Documentation

## Описание
BackButton - это компонент кнопки навигации назад, который позволяет пользователю возвращаться на предыдущий уровень навигации в приложении. Компонент интегрируется с системой навигации и потоком данных приложения.

## Экспортируемые компоненты
- `BackButton` - основной компонент кнопки возврата

## Структура компонента
1. **Кнопка**:
   - CSS классы для стилизации
   - Иконка стрелки влево
   - Текст "Назад"
   - Обработчик клика

## Взаимодействия
- Использует `useNavigationContext` для получения функции возврата (`onBack`)
- Использует `useFlowContext` для получения функций управления узлами и ребрами (`setNodes`, `setEdges`)
- При клике вызывает обработчик `handleClick`, который:
  - Проверяет наличие функции `onBack`
  - Вызывает `onBack` с параметрами `setNodes` и `setEdges`

## Зависимости
- React
- `useNavigationContext` из '../../core/providers/NavigationProvider'
- `useFlowContext` из '../../core/providers/FlowProvider'
- `Icons` из './style/Icons'
- CSS стили из './style/Buttons.css'

## Архитектура
Компонент реализует паттерн функционального компонента React с использованием хуков контекста для интеграции с системой навигации приложения. Компонент принимает опциональные пропсы `label` и `style` для кастомизации, хотя в текущей реализации label не используется. Кнопка визуально оформлена с иконкой стрелки влево и текстом "Назад". Обработчик клика обеспечивает интеграцию с контекстом навигации и потоком данных, передавая функциям управления узлами необходимые параметры для обновления состояния приложения.
